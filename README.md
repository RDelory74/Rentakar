# Rentakar
Exercices SpringBoot microservices, allowing me to use Spring-Boot Framework

## Used Tools 

    - Java 17 
    - Spring Boot v3.3.5
    - Maven 3
    - Spring Data JPA
    - Spring Web MVC
    - Spring Security with JWT
    - database MySQL
